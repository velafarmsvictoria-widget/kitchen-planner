⚙️ One-Time Google Drive Setup

Takes about 5 minutes. Only needs to be done once — then it auto-syncs forever.

1
Go to console.cloud.google.com and sign in with your Gmail.
2
Click New Project → name it Vela Farms Kitchen → Create.
3
APIs & Services → Library → search Google Drive API → Enable.
4
Credentials → Create → OAuth 2.0 Client ID → Web application.
Under Authorised JavaScript origins add your site URL or null for local files.
5
Copy your Client ID. Find YOUR_GOOGLE_CLIENT_ID_HERE in this file and replace it. Save.
💡 This is completely free — Google doesn't charge for this.
